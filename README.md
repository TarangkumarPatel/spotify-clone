# 🎵 Spotify UI Clone

A high-fidelity frontend recreation of the Spotify Web Player interface. This project focuses on mastering complex CSS layouts, positioning, and the "Dark Mode" aesthetic used by world-class streaming platforms.

## 🚀 Project Overview

This repository contains a static clone of the Spotify web interface, built to practice pixel-perfect frontend development. It focuses on the structural layout of a modern media player using clean HTML5 and CSS3.

## ✨ Key UI Components

- **Navigation Sidebar:** A dedicated section for Home and Search functionality with hover effects.
- **Sticky Header:** A functional top navigation bar that remains fixed while scrolling through content.
- **Content Grid:** A responsive container for music cards, featuring hover-state background transitions.
- **Integrated Music Player:** A bottom-docked playback controller with custom-styled progress bars and media icons.

## 🛠️ Tech Stack

- **HTML5:** For semantic document structure.
- **CSS3:** Utilizing Flexbox for layout, custom styling for `range` inputs, and media queries for responsiveness.
- **FontAwesome:** For industry-standard iconography.
- **Google Fonts:** Implementing 'Montserrat' to match the official Spotify branding.
